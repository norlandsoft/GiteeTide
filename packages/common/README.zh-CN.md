# @gitee/tide-common

## 介绍

本包为编辑器提供常用的 `Command`、`Extension`、`Helpers`、`Input Rule` 等。

## 功能

### Extensions

- Commands
  - toggleList
  - sinkListItem
  - liftListItem
  - listBackspace
  - markLeftExit
- LowPriorityKeymap
- HighPriorityKeymap

### Helpers

- isActive
- isMarkActive
- isList
- isListItem
- joinListBackwards
- joinListForwards
- listBackspace
- transformer
- isWindows
- selectionInfo

### Input Rules

- textblockTypeInputRule
- wrappingInputRule
- wrapInListInputRule
